# photo-check
Foto-Check unterstützt bei der datenschutzsensiblen Veröffentlichung von Fotos (Website, Social Media, Print). Anhand weniger Fragen zu Minderjährigen, Erkennbarkeit, Einwilligung und Bildkontext erzeugt das Tool eine strukturierte Empfehlung inklusive Begründung und optionalem HTML-Report. Die Regeln sind vollständig konfigurierbar.
